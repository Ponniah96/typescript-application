Welcome to TypeScript Compiler

Introduction:

TypeScript is a javascript compiler in which codes are strongly typed.

Advantages of TypeScript

  • TypeScript Codes are strongly typed.
  • It is easy to debug, found error while typing code itself in IDE.
  • It is preferable for large scale applications.

Commands to run typeScript compilers

1.Installation

                
    npm i -g typescript

2.In ts.config.json, change below path

                
"rootDir": "./source-typescript",
"outDir": "./source-javascript"

3.Run command to convert JavaScript file

                
tsc.cmd

    value1:string
}

    value2:Number
}

}
